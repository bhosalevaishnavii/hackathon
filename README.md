# HireReady – AI Interview Coach

## Description
HireReady is an AI-powered web application that simulates real interview scenarios using voice interaction and provides instant feedback.

## Features
- AI-generated interview questions
- Voice-based answers
- AI evaluation with feedback and score
- Real-time timer

## Tech Stack
- HTML, CSS, JavaScript
- Gemini API
- Web Speech API

## How to Run
1. Download files
2. Open index.html in browser
3. Add your Gemini API key in script.js
